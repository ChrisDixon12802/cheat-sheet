HTML Cheat Sheet

Basic Structure

<!DOCTYPE html> - Document type declaration
<html> - Root element
<head> - Metadata container
<body> - Visible content
<title> - Page title (in head)

Text Formatting

<h1> to <h6> - Headings (h1 largest)
<p> - Paragraph
<br> - Line break
<hr> - Horizontal rule
<strong> or <b> - Bold text
<em> or <i> - Italic text
<u> - Underlined text
<mark> - Highlighted text
<small> - Smaller text
<del> - Deleted text
<ins> - Inserted text
<sub> - Subscript (H2O)
<sup> - Superscript (x2)

Links & Images

<a href="url">Link</a> - Hyperlink
<a href="url" target="_blank"> - Opens in new tab
<a href="#id"> - Link to element with id
<a href="mailto:email"> - Email link
<img src="image.jpg" alt="description"> - Image
<img width="100" height="50"> - Image dimensions

Lists

Unordered List:
<ul>
  <li>Item</li>
</ul>

Ordered List:
<ol>
  <li>Item</li>
</ol>

Description List:
<dl>
  <dt>Term</dt>
  <dd>Description</dd>
</dl>

Tables

<table> - Table container
<thead> - Table header
<tbody> - Table body
<tfoot> - Table footer
<tr> - Table row
<th> - Header cell
<td> - Data cell
<td colspan="2"> - Spans 2 columns
<td rowspan="2"> - Spans 2 rows

Forms

<form action="/submit" method="POST">
<input type="text" name="username"> - Text input
<input type="password"> - Password input
<input type="email"> - Email input
<input type="number"> - Number input
<input type="date"> - Date picker
<input type="checkbox"> - Checkbox
<input type="radio"> - Radio button
<input type="file"> - File upload
<input type="submit" value="Submit">
<textarea rows="4" cols="50"> - Multi-line text
<select><option>Value</option></select> - Dropdown
<button type="button">Click</button>
<label for="id">Label</label>

Semantic HTML5 Elements

<header> - Page or section header
<nav> - Navigation links
<main> - Main content
<section> - Content section
<article> - Independent content
<aside> - Sidebar content
<footer> - Page or section footer
<figure> - Self-contained content
<figcaption> - Figure caption
<time> - Date/time

Container Elements

<div> - Block-level container
<span> - Inline container
<pre> - Preformatted text
<code> - Code snippet
<blockquote> - Block quotation

Media Elements

<video src="video.mp4" controls></video>
<audio src="audio.mp3" controls></audio>
<iframe src="url"></iframe> - Embed content
<canvas> - Graphics container
<svg> - Scalable Vector Graphics

Common Attributes

id="unique-id" - Unique identifier
class="class-name" - CSS class
style="color: red;" - Inline CSS
title="tooltip text" - Tooltip
data-*="value" - Custom data attribute
hidden - Hide element
disabled - Disable form element
readonly - Read-only input
required - Required form field
placeholder="text" - Input placeholder
alt="description" - Alternative text

Common Meta Tags (in <head>)

<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta name="description" content="Page description">
<meta name="keywords" content="html, css">
<meta name="author" content="Your Name">
<link rel="stylesheet" href="style.css">
<script src="script.js"></script>
<link rel="icon" href="favicon.ico">

HTML Entities (Special Characters)

&lt; - < (less than)
&gt; - > (greater than)
&amp; - & (ampersand)
&quot; - " (quote)
&apos; - ' (apostrophe)
&copy; - © (copyright)
&reg; - ® (registered)
&nbsp; - Non-breaking space
