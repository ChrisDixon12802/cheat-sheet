HTML Cheat Sheet - Forms & Inputs

Form Elements

<form action="/submit" method="POST"> - Form container
<form method="GET"> - GET method (URL params)
<form enctype="multipart/form-data"> - For file uploads
<fieldset> - Groups related form elements
<legend> - Caption for fieldset
<label for="id">Text</label> - Input label

Input Types

<input type="text"> - Single-line text
<input type="password"> - Password field
<input type="email"> - Email validation
<input type="url"> - URL validation
<input type="tel"> - Telephone number
<input type="number" min="0" max="100"> - Number input
<input type="range" min="0" max="100"> - Slider
<input type="date"> - Date picker
<input type="time"> - Time picker
<input type="datetime-local"> - Date and time
<input type="month"> - Month picker
<input type="week"> - Week picker
<input type="color"> - Color picker
<input type="search"> - Search field
<input type="file"> - File upload
<input type="file" multiple> - Multiple files
<input type="hidden"> - Hidden field

Checkboxes & Radio Buttons

Checkbox:
<input type="checkbox" id="agree" name="agree" value="yes">
<input type="checkbox" checked> - Pre-checked

Radio Buttons (same name groups them):
<input type="radio" name="gender" value="male"> Male
<input type="radio" name="gender" value="female"> Female
<input type="radio" checked> - Pre-selected

Button Types

<button type="submit">Submit</button> - Submit form
<button type="button">Click</button> - Generic button
<button type="reset">Reset</button> - Reset form
<input type="submit" value="Submit"> - Submit input
<input type="button" value="Click"> - Button input
<input type="reset" value="Reset"> - Reset input
<button disabled>Disabled</button> - Disabled button

Textarea & Select

Textarea:
<textarea rows="4" cols="50">Text</textarea>
<textarea maxlength="200"> - Character limit
<textarea placeholder="Enter text...">

Select Dropdown:
<select name="options">
  <option value="1">Option 1</option>
  <option value="2" selected>Option 2</option>
</select>

<select multiple> - Multiple selection
<select size="5"> - Visible options

Option Groups:
<optgroup label="Group 1">
  <option>Item</option>
</optgroup>

Form & Input Attributes

name="fieldname" - Field identifier
value="default" - Default value
placeholder="hint" - Placeholder text
required - Field must be filled
readonly - Cannot be modified
disabled - Field is disabled
autofocus - Auto focus on load
autocomplete="off" - Disable autocomplete
maxlength="100" - Max character length
minlength="5" - Min character length
pattern="[A-Z]{3}" - Regex validation
min="0" - Minimum value (number)
max="100" - Maximum value (number)
step="5" - Increment step
multiple - Allow multiple values
accept=".pdf,.jpg" - File types allowed

Datalist (Autocomplete)

<input list="browsers" name="browser">
<datalist id="browsers">
  <option value="Chrome">
  <option value="Firefox">
  <option value="Safari">
</datalist>
