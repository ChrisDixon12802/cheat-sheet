HTML Cheat Sheet - Advanced & Special

HTML Entities (Special Characters)

Common Symbols:
&lt; = < (less than)
&gt; = > (greater than)
&amp; = & (ampersand)
&quot; = " (double quote)
&apos; = ' (apostrophe)
&nbsp; =   (non-breaking space)
&copy; = © (copyright)
&reg; = ® (registered)
&trade; = ™ (trademark)
&euro; = € (euro)
&pound; = £ (pound)
&yen; = ¥ (yen)
&cent; = ¢ (cent)

Mathematical & Arrows

&times; = × (multiplication)
&divide; = ÷ (division)
&plusmn; = ± (plus-minus)
&deg; = ° (degree)
&frac14; = ¼ (one quarter)
&frac12; = ½ (one half)
&frac34; = ¾ (three quarters)
&larr; = ← (left arrow)
&uarr; = ↑ (up arrow)
&rarr; = → (right arrow)
&darr; = ↓ (down arrow)
&harr; = ↔ (left-right arrow)

Container & Grouping Elements

<div> - Block-level container (generic)
<span> - Inline container (generic)
<pre> - Preformatted text (preserves spaces/lines)
<code> - Code snippet
<kbd> - Keyboard input
<samp> - Sample output
<var> - Variable
<blockquote> - Block quotation
<q> - Inline quotation
<cite> - Citation/reference
<abbr title="HyperText Markup Language">HTML</abbr>
<address> - Contact information
<bdo dir="rtl">Text</bdo> - Text direction

Progress & Meter Elements

Progress Bar:
<progress value="70" max="100">70%</progress>
<progress></progress> - Indeterminate progress

Meter (gauge):
<meter value="6" min="0" max="10">6 out of 10</meter>
<meter value="0.6">60%</meter>
<meter low="0.3" high="0.7" optimum="0.8" value="0.5">

Collapsible Content

<details>
  <summary>Click to expand</summary>
  <p>Hidden content here</p>
</details>

<details open> - Start expanded

Dialog Element (Modal)

<dialog id="myDialog">
  <h2>Dialog Title</h2>
  <p>Dialog content</p>
  <button onclick="document.getElementById('myDialog').close()">Close</button>
</dialog>

JavaScript to open:
document.getElementById('myDialog').showModal();

Data Attributes

<div data-user-id="123" data-role="admin">
<button data-action="delete" data-item="5">

Access in JavaScript:
element.dataset.userId (camelCase)
element.getAttribute('data-user-id')

Use data attributes to store custom data on elements

Global Attributes (All Elements)

id="unique-id" - Unique identifier
class="class-name" - CSS classes
style="color: red;" - Inline styles
title="tooltip" - Tooltip text
lang="en" - Language
dir="ltr" - Text direction (ltr/rtl)
tabindex="1" - Tab order
accesskey="h" - Keyboard shortcut
contenteditable="true" - Editable content
draggable="true" - Drag and drop
hidden - Hide element
spellcheck="false" - Spell checking
translate="no" - Translation hint

ARIA (Accessibility)

role="button" - Semantic role
aria-label="Close" - Accessibility label
aria-labelledby="id" - Label reference
aria-describedby="id" - Description reference
aria-hidden="true" - Hidden from screen readers
aria-live="polite" - Dynamic content updates
aria-expanded="false" - Expandable state
aria-checked="true" - Checkbox state
aria-disabled="true" - Disabled state

Output Element

<form oninput="result.value=parseInt(a.value)+parseInt(b.value)">
  <input type="number" id="a" value="0"> +
  <input type="number" id="b" value="0"> =
  <output name="result" for="a b">0</output>
</form>

The <output> element displays calculation results
