HTML Cheat Sheet - Meta & Scripts

Head Section Elements

<head> - Document metadata container
<title> - Page title (required)
<base href="https://example.com/"> - Base URL
<link> - External resources
<meta> - Metadata
<style> - Internal CSS
<script> - JavaScript

Essential Meta Tags

<meta charset="UTF-8"> - Character encoding
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta name="description" content="Page description">
<meta name="keywords" content="html, web, tutorial">
<meta name="author" content="Your Name">
<meta name="robots" content="index, follow">
<meta http-equiv="refresh" content="30"> - Auto refresh
<meta http-equiv="X-UA-Compatible" content="IE=edge">

Open Graph (Social Media)

<meta property="og:title" content="Title">
<meta property="og:description" content="Description">
<meta property="og:image" content="image.jpg">
<meta property="og:url" content="https://example.com">
<meta property="og:type" content="website">
<meta property="og:site_name" content="Site Name">

Twitter Cards:
<meta name="twitter:card" content="summary_large_image">
<meta name="twitter:title" content="Title">
<meta name="twitter:description" content="Description">
<meta name="twitter:image" content="image.jpg">

Link Elements

<link rel="stylesheet" href="style.css"> - CSS
<link rel="icon" href="favicon.ico"> - Favicon
<link rel="apple-touch-icon" href="icon.png">
<link rel="canonical" href="https://example.com">
<link rel="alternate" href="page.html" hreflang="es">
<link rel="preload" href="font.woff2" as="font">
<link rel="prefetch" href="page2.html">
<link rel="dns-prefetch" href="//example.com">
<link rel="manifest" href="manifest.json">

Script Elements

External Script:
<script src="script.js"></script>
<script src="script.js" defer></script> - Execute after parsing
<script src="script.js" async></script> - Execute asynchronously
<script type="module" src="module.js"></script> - ES6 module
<script nomodule src="fallback.js"></script> - Fallback

Inline Script:
<script>
  console.log('Hello World');
</script>

Script Placement:
• End of body - Better page load
• Head with defer - Modern approach
• Head with async - Independent scripts

Style Elements

Internal CSS:
<style>
  body { background: #fff; }
  .class { color: blue; }
</style>

Scoped Styles (modern):
<style scoped> - Applies only to parent element

Media Queries:
<style media="screen and (max-width: 768px)">

Noscript Element

<noscript>
  <p>JavaScript is disabled. Please enable it.</p>
</noscript>

Shows content when JavaScript is disabled in the browser.

HTML Comments

<!-- This is a comment -->

<!-- Multi-line comment
     Can span multiple lines
-->

Conditional Comments (IE only):
<!--[if IE]>
  <p>Only visible in Internet Explorer</p>
<![endif]-->

Character Sets

<meta charset="UTF-8"> - Unicode (recommended)
<meta charset="ISO-8859-1"> - Latin alphabet
<meta charset="windows-1252"> - Windows encoding

Always use UTF-8 for best international character support.
