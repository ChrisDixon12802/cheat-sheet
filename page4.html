HTML Cheat Sheet - Semantic & Media

Semantic HTML5 Elements

<header> - Page/section header
<nav> - Navigation links
<main> - Main content (unique per page)
<section> - Thematic content section
<article> - Independent, self-contained content
<aside> - Sidebar/tangential content
<footer> - Page/section footer
<figure> - Self-contained media
<figcaption> - Caption for figure
<details> - Collapsible details
<summary> - Heading for details
<mark> - Marked/highlighted text
<time datetime="2025-12-18"> - Date/time

Semantic Page Structure

<body>
  <header>
    <nav>Navigation</nav>
  </header>

  <main>
    <article>
      <section>Content</section>
    </article>
    <aside>Sidebar</aside>
  </main>

  <footer>Footer</footer>
</body>

Images & Picture

<img src="image.jpg" alt="Description">
<img width="300" height="200"> - Fixed size
<img loading="lazy"> - Lazy loading
<img srcset="small.jpg 480w, large.jpg 800w"> - Responsive
<img usemap="#mapname"> - Image map

Picture Element (responsive images):
<picture>
  <source media="(min-width: 800px)" srcset="large.jpg">
  <source media="(min-width: 400px)" srcset="medium.jpg">
  <img src="small.jpg" alt="Description">
</picture>

Image Maps

<img src="map.jpg" usemap="#workmap">
<map name="workmap">
  <area shape="rect" coords="34,44,270,350" href="link.html">
  <area shape="circle" coords="337,300,44" href="link2.html">
  <area shape="poly" coords="140,121,181,116,204" href="link3.html">
</map>

Shape types:
shape="rect" - Rectangle (x1,y1,x2,y2)
shape="circle" - Circle (x,y,radius)
shape="poly" - Polygon (x1,y1,x2,y2,x3,y3...)

Audio

<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  <source src="audio.ogg" type="audio/ogg">
  Your browser does not support audio.
</audio>

Audio Attributes:
<audio controls> - Show controls
<audio autoplay> - Auto play
<audio loop> - Loop playback
<audio muted> - Start muted
<audio preload="auto"> - Preload audio

Video

<video width="640" height="360" controls>
  <source src="video.mp4" type="video/mp4">
  <source src="video.webm" type="video/webm">
  <track src="subtitles.vtt" kind="subtitles" srclang="en">
  Your browser does not support video.
</video>

Video Attributes:
<video controls> - Show controls
<video autoplay> - Auto play
<video loop> - Loop playback
<video muted> - Start muted
<video poster="thumbnail.jpg"> - Thumbnail image
<video preload="metadata"> - Preload setting

Canvas & SVG

Canvas (JavaScript drawing):
<canvas id="myCanvas" width="200" height="100">
  Fallback content
</canvas>

SVG (Scalable Vector Graphics):
<svg width="100" height="100">
  <circle cx="50" cy="50" r="40" fill="red" />
</svg>

iFrame (Embed Content)

<iframe src="https://example.com"></iframe>
<iframe width="560" height="315"> - Set dimensions
<iframe frameborder="0"> - No border
<iframe allowfullscreen> - Allow fullscreen
<iframe loading="lazy"> - Lazy loading
<iframe sandbox> - Security restrictions
<iframe name="framename"> - Target name
