HTML Cheat Sheet - Tables & Lists

Table Structure

<table> - Table container
<thead> - Table header section
<tbody> - Table body section
<tfoot> - Table footer section
<tr> - Table row
<th> - Header cell
<td> - Data cell
<caption> - Table caption
<colgroup> - Column group
<col> - Column properties

Complete Table Example

<table border="1">
  <caption>Student Grades</caption>
  <thead>
    <tr>
      <th>Name</th>
      <th>Grade</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>John</td>
      <td>A</td>
    </tr>
  </tbody>
</table>

Table Cell Attributes

<td colspan="2"> - Spans 2 columns
<td rowspan="3"> - Spans 3 rows
<th scope="col"> - Header for column
<th scope="row"> - Header for row
<table border="1"> - Table border
<table cellpadding="10"> - Cell padding
<table cellspacing="5"> - Cell spacing
<td align="center"> - Horizontal alignment
<td valign="top"> - Vertical alignment

Unordered Lists

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

List Style Types:
<ul type="disc"> - Filled circle (default)
<ul type="circle"> - Empty circle
<ul type="square"> - Square bullet
<ul style="list-style-type: none;"> - No bullets

Ordered Lists

<ol>
  <li>First</li>
  <li>Second</li>
  <li>Third</li>
</ol>

List Types:
<ol type="1"> - Numbers (default)
<ol type="A"> - Uppercase letters
<ol type="a"> - Lowercase letters
<ol type="I"> - Roman numerals (uppercase)
<ol type="i"> - Roman numerals (lowercase)
<ol start="5"> - Start from number 5
<ol reversed> - Reverse order

Description Lists

<dl>
  <dt>Term 1</dt>
  <dd>Description 1</dd>
  <dt>Term 2</dt>
  <dd>Description 2</dd>
</dl>

<dl> - Description list
<dt> - Term/name
<dd> - Description/definition

Nested Lists

<ul>
  <li>Item 1
    <ul>
      <li>Sub-item 1</li>
      <li>Sub-item 2</li>
    </ul>
  </li>
  <li>Item 2</li>
</ul>
